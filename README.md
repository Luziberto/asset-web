This is the project for the `Crypto check` website.

## Requirements

Make sure you have all this installed:

- `node`
- `yarn` or `npm`

## Stack

This project uses the following stack:

- Node 16.x.x
- Vue 3
- Typescript
- Vite
- Pinia
- Tailwind
- EsLint

## How to run

To run the project follow these steps:

With yarn

```sh
$ yarn i
$ yarn run dev
```

Or with npm

```sh
$ npm i
$ npm run dev
```
